# FurstSpark Tool

Minimal Vite + React setup for local development.

## Quick start

1. Install dependencies:

   npm install

2. Run dev server:

   npm run dev

3. Open the URL shown in the terminal (default http://localhost:5173)

Notes:
- The project uses the Tailwind CDN in `index.html` for styling.
- If you plan to use Tailwind with PostCSS, install and configure it separately.
